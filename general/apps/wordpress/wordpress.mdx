
<Info>
    This page explains the steps you need to follow to set up the WordPress connection in VocbotAI. If any of the steps are outdated, please let us know!
</Info>

1. Add your WordPress public URL (without any path in the address) in the **WordPress instance URL** field on VocbotAI.
1. Click **Submit** button on VocbotAI.
1. Congrats! Start using your new WordPress connection within the flows.
