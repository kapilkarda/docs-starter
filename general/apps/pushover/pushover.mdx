

1. Login to [your account page](https://pushover.net/login) on Pushover.
2. Copy the **Your User Key** value to the **User Key** field in VocbotAI connection page.
3. Create a new application from [here](https://pushover.net/apps/build) on Pushover.
4. Copy the **API Token/Key** value to the **API Token** field in VocbotAI connection page.
5. Write any screen name to be displayed in VocbotAI.
6. Click `Submit`.
7. Start using Pushover integration with VocbotAI!
